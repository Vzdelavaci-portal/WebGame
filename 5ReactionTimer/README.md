# 🎮 WebGame #5 – Reaction Timer

---

## 🇬🇧 English

Simple browser game built with **HTML, CSS and JavaScript**.

### 🎮 About the game
Test your reaction speed by clicking as fast as possible when the screen turns green.

- Click to start  
- Wait for the green screen  
- Click as fast as possible  
- Try not to click too early 😅  

### 🚀 How to run
1. Download or clone the repository  
2. Open the file:  
   reaction-timer.html  
3. Play directly in your browser  

### 🛠 Technologies
- HTML  
- CSS  
- JavaScript (Vanilla)  

### 📌 Features
- Random delay before action  
- Reaction time measurement (ms)  
- Early click detection  
- Color-based feedback  
- Restart option  
- Simple and clean UI

## 🇨🇿 Česky

### 🎮 O hře
Otestuj svou reakční rychlost – klikni co nejrychleji, jakmile se obrazovka změní na zelenou.

- Klikni pro start  
- Počkej na zelenou barvu  
- Klikni co nejrychleji  
- Nezrychluj se moc – předčasný klik = chyba 😅

1. Stáhni nebo naklonuj repozitář

   reaction-timer.html
