# 🎮 WebGame #13 – Mini Snake

---

## 🇬🇧 English

Simple browser game built with **HTML, CSS and JavaScript**.

### 🎮 About the game
Control the snake, collect food, and try to survive as long as possible.

- Use **arrow keys** to control the snake  
- Eat food to grow and increase your score 🍎  
- Avoid hitting the walls or yourself  
- The longer you survive, the higher your score  

### 🚀 How to run
1. Download or clone the repository  
2. Open the file:  
   mini-snake.html  
3. Play directly in your browser  

### 🛠 Technologies
- HTML  
- CSS  
- JavaScript (Vanilla)  

### 📌 Features
- Classic snake gameplay  
- Keyboard controls  
- Score tracking  
- Random food generation  
- Collision detection (walls + self)  
- Restart functionality  
- Simple and clean UI

## 🇨🇿 Česky

Jednoduchá webová hra vytvořená pomocí **HTML, CSS a JavaScriptu**.

### 🎮 O hře
Ovládej hada, sbírej jídlo a snaž se přežít co nejdéle.

- Ovládání pomocí **šipek**  
- Sbírej jídlo a zvyšuj skóre 🍎  
- Vyhýbej se stěnám a vlastnímu tělu  
- Čím déle vydržíš, tím vyšší skóre získáš

1. Stáhni nebo naklonuj repozitář  
2. Otevři soubor:  
   mini-snake.html
