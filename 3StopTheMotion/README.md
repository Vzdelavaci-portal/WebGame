# 🎮 WebGame #3 – Stop the Motion

---

## 🇬🇧 English

Simple browser game built with **HTML, CSS and JavaScript**.

### 🎮 About the game
Your goal is to stop the moving object as close to the center (target zone) as possible.

- Click the **STOP** button at the right moment  
- The closer to the center, the higher your score  
- The game gets faster after each round  

### 🚀 How to run
1. Download or clone the repository  
2. Open the file:  
   stop-the-motion.html  
3. Play directly in your browser  

### 🛠 Technologies
- HTML  
- CSS  
- JavaScript (Vanilla)  

### 📌 Features
- Smooth object movement  
- Accuracy-based scoring system  
- Increasing difficulty  
- Restart option  
- Simple and clean UI

## 🇨🇿 Česky

Jednoduchá webová hra vytvořená pomocí **HTML, CSS a JavaScriptu**.

### 🎮 O hře
Tvým cílem je zastavit pohybující se objekt co nejblíže středu (cílové zóny).

- Klikni na tlačítko **STOP** ve správný moment  
- Čím blíže středu, tím více bodů získáš  
- Hra se po každém kole zrychluje  

### 🚀 Jak spustit
1. Stáhni nebo naklonuj repozitář  
2. Otevři soubor:  
   stop-the-motion.html  
3. Hraj přímo v prohlížeči
