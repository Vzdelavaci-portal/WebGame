# 🎮 WebGame #4 – Jump Over Obstacles

---

## 🇬🇧 English

Simple browser game built with **HTML, CSS and JavaScript**.

### 🎮 About the game
Your goal is to jump over incoming obstacles and survive as long as possible.

- Press **SPACE** or click the button to jump  
- Avoid hitting obstacles  
- The longer you survive, the higher your score  
- The game speeds up over time  

### 🚀 How to run
1. Download or clone the repository  
2. Open the file:  
   jump-over-obstacles.html  
3. Play directly in your browser  

### 🛠 Technologies
- HTML  
- CSS  
- JavaScript (Vanilla)  

### 📌 Features
- Jump mechanic with gravity  
- Collision detection  
- Increasing difficulty  
- Score system  
- Restart option  
- Keyboard + button controls

## 🇨🇿 Česky

Jednoduchá webová hra vytvořená pomocí **HTML, CSS a JavaScriptu**.

### 🎮 O hře
Tvým cílem je přeskakovat překážky a vydržet co nejdéle.

- Stiskni **mezerník** nebo tlačítko pro skok  
- Vyhýbej se překážkám  
- Čím déle vydržíš, tím vyšší skóre získáš  
- Hra se postupně zrychluje  

### 🚀 Jak spustit
1. Stáhni nebo naklonuj repozitář  
2. Otevři soubor:  
   jump-over-obstacles.html  
3. Hraj přímo v prohlížeči  

### 🛠 Technologie
- HTML  
- CSS  
- JavaScript (bez knihoven)  

### 📌 Funkce
- Skákání s gravitací
