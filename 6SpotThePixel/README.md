# 🎮 WebGame #6 - Spot the Pixel

---

## 🇬🇧 English

Simple browser game built with **HTML, CSS and JavaScript**.

### 🎮 About the game
Your goal is to find the slightly different pixel in the grid.

- Click on the pixel that looks different  
- Each level gets harder  
- One wrong click = game over 😅  

### 🚀 How to run
1. Download or clone the repository  
2. Open the file:  
   spot-the-pixel.html  
3. Play directly in your browser  

### 🛠 Technologies
- HTML  
- CSS  
- JavaScript (Vanilla)  

### 📌 Features
- Dynamic grid size (increasing difficulty)  
- Random color generation  
- Subtle color differences  
- Score system  
- Level progression  
- Simple and clean UI
