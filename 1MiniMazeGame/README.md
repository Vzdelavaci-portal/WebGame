# 🧩 Mini Maze Game

## English version

Simple browser game built with **HTML, CSS and JavaScript**.

### 🎮 About the game

Your goal is to move the green player through the maze and reach the finish.

* Use **arrow keys** or on-screen buttons
* Avoid walls
* Reach the finish 🏁

### 🚀 How to run

1. Download or clone the repository
2. Open the file:

```
mini-maze.html
```

3. Play directly in your browser

### 🛠 Technologies

* HTML
* CSS
* JavaScript (Vanilla)

### 📌 Features

* Keyboard controls
* Mobile-friendly buttons
* Simple maze logic
* Restart button
