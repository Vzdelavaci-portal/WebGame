# 🎮 WebGame #2 – Catch the Object

---

## 🇬🇧 English

Simple browser game built with **HTML, CSS and JavaScript**.

### 🎮 About the game
Your goal is to catch the falling object using the blue platform.

- Move left and right using **arrow keys** or buttons  
- Catch the falling object to gain points  
- Avoid missing it – you have limited lives  

### 🚀 How to run
1. Download or clone the repository  
2. Open the file:  
   catch-the-object.html  
3. Play directly in your browser  

### 🛠 Technologies
- HTML  
- CSS  
- JavaScript (Vanilla)  

### 📌 Features
- Player movement (keyboard + mobile buttons)  
- Falling object mechanics  
- Score system  
- Lives system  
- Increasing difficulty  
- Restart option

## 🇨🇿 Česky

Jednoduchá webová hra vytvořená pomocí **HTML, CSS a JavaScriptu**.

### 🎮 O hře
Tvým cílem je zachytit padající objekt pomocí modré plošinky.

- Pohybuj se doleva a doprava pomocí **šipek** nebo tlačítek  
- Chyť objekt a získej body  
- Nenech ho spadnout – máš omezený počet životů  

### 🚀 Jak spustit
1. Stáhni nebo naklonuj repozitář  
2. Otevři soubor:  
   catch-the-object.html  
3. Hraj přímo v prohlížeči
