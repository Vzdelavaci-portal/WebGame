# 🎮 WebGame #7 – Click Before It Disappears

---

## 🇬🇧 English

Simple browser game built with **HTML, CSS and JavaScript**.

### 🎮 About the game
Your goal is to click the target before it disappears.

- A target appears randomly on the screen  
- Click it before it disappears  
- Each successful hit increases your score  
- The game gets faster over time  
- Missing too many targets = game over 😅  

### 🚀 How to run
1. Download or clone the repository  
2. Open the file:  
   click-before-it-disappears.html  
3. Play directly in your browser  

### 🛠 Technologies
- HTML  
- CSS  
- JavaScript (Vanilla)  

### 📌 Features
- Random target positioning  
- Increasing difficulty (faster disappearance)  
- Score system  
- Lives system  
- Miss detection  
- Restart functionality  
- Simple and clean UI
