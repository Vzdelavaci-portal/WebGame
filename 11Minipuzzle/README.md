# 🎮 WebGame #11 – Mini Puzzle

---

## 🇬🇧 English

Simple browser puzzle game built with **HTML, CSS and JavaScript**.

### 🎮 About the game
Rearrange the tiles to solve the puzzle in the correct order.

- Click tiles to move them into the empty space  
- Arrange numbers from **1 to 8** in the correct order  
- Try to solve it in as few moves as possible  
- Challenge your logic and thinking 🧠  

### 🚀 How to run
1. Download or clone the repository  
2. Open the file:  
   mini-puzzle.html  
3. Play directly in your browser  

### 🛠 Technologies
- HTML  
- CSS  
- JavaScript (Vanilla)  

### 📌 Features
- Shuffle system (always solvable puzzle)  
- Move counter  
- Win detection  
- Simple and clean UI  
- Restart functionality
