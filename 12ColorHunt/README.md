# 🎮 WebGame #12 – Color Hunt

---

## 🇬🇧 English

Simple browser game built with **HTML, CSS and JavaScript**.

### 🎮 About the game
Test your brain and reaction speed in this color-based challenge.

- A color name is displayed on the screen  
- The text color may not match the word  
- Click the correct color based on the **text**, not the color  
- This creates a classic brain-confusing effect (Stroop effect) 😄  

### 🚀 How to run
1. Download or clone the repository  
2. Open the file:  
   color-hunt.html  
3. Play directly in your browser  

### 🛠 Technologies
- HTML  
- CSS  
- JavaScript (Vanilla)  

### 📌 Features
- Random color combinations  
- Stroop effect gameplay  
- Timer system  
- Score tracking  
- Instant feedback  
- Restart functionality  
- Simple and clean UI
