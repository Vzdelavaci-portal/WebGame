# 🎮 WebGame #9 – Color Memory

---

## 🇬🇧 English

Simple browser game built with **HTML, CSS and JavaScript**.

### 🎮 About the game
Test your memory by repeating an increasing sequence of colors.

- Watch the sequence of colors  
- Repeat it in the correct order  
- Each round adds a new color  
- One mistake = game over 😅  

### 🚀 How to run
1. Download or clone the repository  
2. Open the file:  
   color-memory.html  
3. Play directly in your browser  

### 🛠 Technologies
- HTML  
- CSS  
- JavaScript (Vanilla)  

### 📌 Features
- Random color sequence  
- Increasing difficulty  
- Memory-based gameplay  
- Level progression  
- Simple and clean UI
