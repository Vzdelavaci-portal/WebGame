# 🎮 WebGame #8 – Math Challenge

---

## 🇬🇧 English

Simple browser game built with **HTML, CSS and JavaScript**.

### 🎮 About the game
Test your math skills by solving as many problems as possible before time runs out.

- Solve math problems (addition, subtraction, multiplication)  
- Type your answer and press **Enter**  
- Each correct answer increases your score  
- You have limited time ⏱️  

### 🚀 How to run
1. Download or clone the repository  
2. Open the file:  
   math-challenge.html  
3. Play directly in your browser  

### 🛠 Technologies
- HTML  
- CSS  
- JavaScript (Vanilla)  

### 📌 Features
- Random math problem generation  
- Multiple operations (+, -, *)  
- Timer system  
- Score tracking  
- Instant feedback (correct / wrong)  
- Restart functionality  
- Simple and clean UI
