# 🎮 WebGame #10 – Click Survival

---

## 🇬🇧 English

Simple browser game built with **HTML, CSS and JavaScript**.

### 🎮 About the game
Test your speed and focus in this fast-paced clicking survival game.

- Click the **green targets** to gain points 🎯  
- Avoid clicking the **red targets** ❌  
- Don’t miss too many targets  
- The game gets faster over time ⚡  
- Survive as long as possible  

### 🚀 How to run
1. Download or clone the repository  
2. Open the file:  
   click-survival.html  
3. Play directly in your browser  

### 🛠 Technologies
- HTML  
- CSS  
- JavaScript (Vanilla)  

### 📌 Features
- Random target spawning  
- Good vs bad targets (green/red)  
- Increasing difficulty (speed)  
- Score system  
- Lives system  
- Miss detection  
- Restart functionality  
- Simple and clean UI
